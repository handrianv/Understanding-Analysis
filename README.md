The book: [Understanding Analysis, 2nd Edition by Stephen Abbott](https://link.springer.com/book/10.1007/978-1-4939-2712-8). Also check out its [MAA review](https://old.maa.org/press/maa-reviews/understanding-analysis). This repo contains my solutions to the exercises.

If you stumbled upon this page, most likely you're looking for a solution manual. Unfortunately this repo is not a good one. I wrote my full thought process for each problem, so the solutions are much more verbose than necessary. I also recorded all my wrong attempts, and there are lots of them. The book has an official solution manual for odd-numbered exercises.

But if you still wish to use this anyway, then feel free to do so, and I'd greatly appreciate any suggestions or improvements. This is my first time tackling a pure math book. Happy solving!
